## 1️⃣ 試合前夜の睡眠戦略（2026-05-22 目安）

### Frontmatter 案

```yaml
title: 試合前夜に眠れない…救急医が教える「副交感神経を整える」就寝術
slug: 2026-05-22-pre-match-sleep-strategy
category: 医学コラム
tags: [睡眠, インターハイ, メンタル, 自律神経, 高校サッカー]
description: インターハイ予選を控えた選手が「緊張で眠れない」のは自然な反応です。救急医の視点から、副交感神経を優位にする就寝3時間前の準備、寝室環境、起き上がり方までを医学的に解説します。
```

### 狙うキーワード

`試合前 眠れない` / `スポーツ 緊張 不眠` / `インターハイ 前日 過ごし方` / `アスリート 睡眠`

### セクション構成

1. **導入**：救急現場で見た「徹夜明け試合 → 熱中症や脱水で搬送」のパターン
2. **なぜ試合前夜は眠れないのか**：交感神経優位・コルチゾール上昇の生理学
3. **就寝3時間前から始める準備**：食事・入浴・スマホ・カフェイン
4. **寝室環境のチェックリスト**：温度18-20℃・湿度50%・遮光・無音
5. **それでも眠れない時の最終手段**：横になっているだけでも回復はある
6. **当日朝の整え方**：起床光浴・タンパク質・歩行
7. **もし徹夜状態で試合があるとき**：パワーナップとカフェインの戦略的使用

### 現場感のある語りどころ

- 「緊張で眠れなかった選手が脱水＋低血糖で熱中症発症」というケース
- 「眠れないことを気にして余計眠れない」スパイラルへの言及

### 内部リンク候補

- 熱中症記事（睡眠不足は熱中症リスクを上げる文脈で）


### 狙うキーワード

`サッカー 貧血` / `運動 疲れやすい 中高生` / `鉄欠乏 スポーツ` / `フェリチン アスリート`

### セクション構成

1. **導入**：「成績が落ちた、気合いが足りない」の前に疑うこと
2. **なぜサッカー選手に鉄欠乏が多いか**：汗・足底打撲(溶血)・腸管出血・成長期需要
3. **隠れた貧血のサイン**：朝の倦怠感・氷をかじる(氷食症)・爪の変形・易疲労性
4. **病院で測る検査**：ヘモグロビン だけでなく **フェリチン** を見る重要性
5. **食事での鉄分摂取**：ヘム鉄・非ヘム鉄・吸収率を上げるビタミンC
6. **サプリメントの正しい使い方**：医師の判断で使うべき理由・副作用
7. **治療開始から競技復帰まで**：3ヶ月の改善過程と再検査タイミング

### 現場感のある語りどころ

- 救急で「失神」搬送されてきて鉄欠乏が判明したケース
- 「気合いで克服しようとして悪化」した部活生の話


### 狙うキーワード

`足首捻挫 サッカー` / `RICE処置 やり方` / `捻挫 早く治す` / `捻挫 病院 行くべき`

### セクション構成

1. **導入**：救急で見る「軽く考えて悪化させた捻挫」のパターン
2. **RICE処置の正しいやり方**：Rest・Icing・Compression・Elevation の各論
3. **やってはいけない民間療法TOP5**：温める・マッサージ・無理して歩く 等
4. **テーピングで試合続行は OK か NG か**：状況別判断
5. **病院に行くべき判断基準**：靱帯断裂・骨折を疑うサイン
6. **復帰までのステップ**：段階的プロトコル(免荷→部分荷重→ジョグ→競技)
7. **再発予防**：足関節周囲筋トレ・バランスボード

### 現場感のある語りどころ

- 「軽い捻挫を放置して半年後にも痛みが残るケース」
- 整形外科に行くべきかどうかの判断ライン

### 狙うキーワード

`スポーツドリンク 違い` / `経口補水液 OS-1 サッカー` / `水分補給 タイミング` / `部活 熱中症 飲み物`

1. **導入**：5月記事のおさらい + 真夏編の位置づけ


### 狙うキーワード

`サッカー 脳震盪` / `頭部外傷 復帰` / `セカンドインパクト症候群` / `スポーツ脳震盪 SCAT5`

### セクション構成

1. **導入**：救急で見た重症頭部外傷からの教訓
2. **脳震盪とは何か**：医学的定義と「軽い意識消失」の誤解
3. **現場での見極めチェックリスト**：意識状態・記憶・平衡感覚・症状
4. **セカンドインパクト症候群の恐怖**：脳震盪後に再受傷で死亡する若年スポーツ選手
5. **復帰までの段階的プログラム(GRTP)**：6段階で最低6日間
6. **ヘディング指導の医学的観点**：年齢別の推奨と慢性的影響
7. **指導者・保護者・チームメイトの責任**：本人は判断できない前提で

### 現場感のある語りどころ

- 「軽い脳震盪と思って続行 → 数日後に致死性脳浮腫」の若年症例
- 救急現場でCTを撮って分かる「実は出血があった」ケース
- 試合中に脳震盪と分からない症状(短時間の言動変化など)

### セクション構成

1. **導入**：「夏に追い込む」文化の落とし穴
2. **オーバートレーニング症候群とは**：単なる疲労との違い

7. **「休む勇気」と指導者の役割**：休息も練習の一部

## 📝 執筆時の共通ルール（ブランド維持のため）

### トーン
- **柔らかい語りかけ**：「〜していきましょう」「お伝えしたい」
- **救急医視点を盛り込む**：1記事に1〜2回、現場のエピソード
- **専門用語は必ず日本語訳**：例「フェリチン(貯蔵鉄)」

- 導入(200-300字) + 6〜7セクション + まとめ
- 各セクションは H2(##)、必要に応じて H3(###)
- 表(`|`)・箇条書きを適切に使う
- 全体 2,000〜3,000字目安

- description は 120〜160字
- tags は 5個前後
- 内部リンクは 2〜5本
- canonical URL は自動生成される

- 公開日に5ツイートのスレッド
