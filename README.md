# 高校サッカー順位確認システム

男子サッカーの高円宮杯U-18（プレミアリーグ、プリンスリーグ、都道府県リーグ）、全国高校サッカー選手権、インターハイの順位と成績を都道府県別に確認できるWebサイトです。

## 📋 主な機能

### 🗾 インタラクティブな日本地図
- 47都道府県をクリックして詳細情報を表示
- リーグレベルに応じた色分け表示
  - 🥇 ゴールド：プレミアリーグ所属チームあり
  - 🥈 ブルー：プリンスリーグ所属チームあり
  - 🥉 ライトブルー：都道府県リーグのみ
  - ⚪ グレー：データなし

### 📊 都道府県別詳細表示
- **リーグ戦タブ**
  - 県内順位、チーム名、所属リーグ
  - 順位、勝点、試合数、勝敗分、得失点
  - リーグレベルに応じたバッジ表示
  - 県内トップ3には特別な順位バッジ

- **大会成績タブ**
  - 全国高校サッカー選手権の成績
  - インターハイの成績
  - 年度、出場チーム、結果

### 🔍 チーム検索機能
- チーム名または高校名でリアルタイム検索
- 検索結果から該当都道府県の詳細を即座に表示

### ⚙️ 管理画面
- **チーム管理**
  - チームの新規追加、編集、削除
  - リーグ順位、勝点、試合成績の入力
  - 県内順位の管理

- **大会成績管理**
  - 選手権・インターハイの成績追加
  - 出場チーム、年度、結果の記録

- **データ管理**
  - データのエクスポート（JSON形式）
  - データのインポート（バックアップから復元）
  - デフォルトデータへのリセット

## 🚀 使い方

### 1. サイトを開く

プロジェクトフォルダ内の `index.html` をブラウザで開きます。

```bash
# ローカルサーバーで起動（推奨）
# Python 3の場合
python -m http.server 8000

# Node.jsの場合
npx http-server
```

ブラウザで `http://localhost:8000` を開きます。

### 2. 都道府県の情報を確認

1. トップページの日本地図で確認したい都道府県をクリック
2. モーダルウィンドウが開き、その都道府県のチーム情報が表示されます
3. 「リーグ戦」と「大会成績」のタブを切り替えて情報を確認できます

### 3. チームを検索

1. ヘッダーの「🔍 検索」ボタンをクリック
2. 検索ボックスにチーム名または高校名を入力
3. 該当するチームが表示されるので、クリックすると詳細が表示されます

### 4. データを更新（管理画面）

1. ヘッダーの「⚙️ 管理」ボタンをクリックして管理画面を開く
2. **チーム管理**タブ:
   - 都道府県を選択
   - 「新規チーム追加」ボタンをクリック
   - チーム情報を入力して保存
   - 既存チームは「編集」「削除」ボタンで管理

3. **大会成績管理**タブ:
   - 都道府県を選択
   - 「新規大会成績追加」ボタンをクリック
   - 年度、大会名、チーム、結果を入力して保存

4. **データ管理**タブ:
   - 「データをエクスポート」：現在のデータをJSONファイルとしてダウンロード
   - 「データをインポート」：バックアップファイルからデータを復元
   - 「デフォルトに戻す」：初期サンプルデータに戻す

## 📊 データについて

### チーム数
- **全47都道府県**: 各都道府県に**約10チーム**のデータが含まれています
- **合計**: 約470チーム以上

### データ構成
各都道府県には以下のチームが含まれます：
- プレミアリーグ・プリンスリーグ所属の実在チーム
- 都道府県リーグの主要チーム
- 自動生成された地域の高校・ユースチーム

### データの編集
管理画面から自由に追加・編集・削除できます。編集したデータはブラウザに保存され、次回アクセス時も保持されます。

## 🔧 カスタマイズ

### 実在チーム名による10チーム構成

このシステムは**実在の高校・ユースチーム名パターン**を使用しています：

- 全47都道府県に**10チームずつ**登録
- プレミアリーグ・プリンスリーグ所属チームは**実際のデータ**を使用
- 県リーグチームは各都道府県の**一般的なチーム名パターン**を使用
  - 例: 「東京東高校」「大阪商業高校」「福岡FC U-18」など
- 成績データは県内順位に応じて現実的な数値を生成

以下の都道府県は**2025年度の実際のリーグ順位データ**が10チーム登録済み：

- **静岡県**: 静岡学園高校(プレミアWEST 10位) + T1リーグ・静岡県1部実順位

#### その他の都道府県
残り35都道府県は、プレミア・プリンスリーグ所属の実在チーム（2-3チーム）+ 各県の実在高校名ベースのチーム構成で、10チームを達成しています。

### チームの追加・編集

1. 管理画面の「チーム管理」から都道府県を選択
2. 「新規チーム追加」でチーム情報を入力
3. リーグ情報、成績を入力して保存
4. 編集・削除も簡単に行えます

📖 **他の主要都道府県に実在チームを追加したい場合**  
→ [ADDING_TEAMS.md](ADDING_TEAMS.md) に追加ガイドと主要チームリストがあります

## 🔍 トラブルシューティング

### ⚠️ 重要：古いデータが表示される場合

**47都道府県ではなく5都道府県しか表示されない場合**は、ブラウザのローカルストレージに古いデータが残っています。

👉 **詳しい解決方法は [TROUBLESHOOTING.md](TROUBLESHOOTING.md) をご覧ください**

#### 簡単な解決法
1. ページを一度閉じて、**再度開く**
2. **強制再読み込み**: `Ctrl + Shift + R`（Mac: `Cmd + Shift + R`）
3. それでも解決しない場合：
   - ブラウザの開発者ツール（F12）を開く
   - Application → Local Storage → すべてクリア
   - ページを再読み込み

### 地図が表示されない

- ブラウザのJavaScriptが有効か確認
- コンソールエラーをチェック

### データが保存されない

- ブラウザのローカルストレージが有効か確認
- プライベートブラウジングモードでは保存できません
