## 🎯 戦術的な見どころ（TACTICAL）

### リーグ全体の構図（第8節終了時点）

優勝争いは **流通経済大柏（首位）・鹿島アントラーズユース（2位）・FC東京U-18（3位）** の三つ巴。この3チームはそれぞれ全く異なる「強さの方程式」を持っており、対戦カードごとの戦術的噛み合わせが勝敗を分けている。

| クラスター | チーム | 特徴 |
|---|---|---|
| 上位 | 流経大柏・鹿島ユース・FC東京U-18 | 異なる「強さの方程式」を競う |
| 中位 | 柏U-18・前橋育英・帝京長岡・東京V | 戦術的尖りで上位に挑戦 |
| 下位 | 青森山田・川崎F U-18・仙台U・昌平・横浜FC | 構造的変革期 |

### トップ3の「強さの方程式」

**🛡️ 流通経済大柏（21pts／7勝1敗／13得点-4失点）**
榎本雅大監督が築く **「新しい流経」**。8試合わずか4失点の堅守と、U-18日本代表DFメンディーサイモン友を軸とした最終ラインが盤石。1-0・2-0で勝ち切る試合運びが特徴で、FW渡辺瞳也（8得点）が限られたチャンスを確実に仕留める。鹿島ユースに1-3で唯一敗北、それ以外は7連勝。

**⚔️ 鹿島アントラーズユース（18pts／6勝2敗／24得点-11失点）**
8試合24得点の圧倒的火力。U-17日本代表に **元砂晏翔仁ウデンバ（キャプテン）・大下幸誠・倉橋幸暉・岩土そら・髙木瑛人** の5名を同時送り込む「ミニ・ナショナルチーム」状態。FW吉田湊海がリーグ得点王（9得点）を独走。U-18でも大貫琉偉が代表入り。一方で守備の連携に脆さがあり、柏U-18に0-2、帝京長岡に2-3など失点11。

**⚙️ FC東京U-18（16pts／5勝1分2敗／17得点-8失点）**
U-17日本代表で背番号10を背負うMF北原槙が中盤の創造性を担い、FW樋口佳（8得点）と「北原-樋口」ホットラインがリーグ屈指の戦術的兵器。GK渡邊麻舟もU-18代表で、**センターライン全体に代表級**が並ぶ稀有なバランス型。

### 中位の戦術的多様性

- **柏レイソルU-18（4位）**：志田達郎監督の緻密な守備組織。リーグ2位の5失点。鹿島を2-0で完封するなど対戦相手のストロングを消す戦術眼が光る。攻撃の活性化が課題
- **前橋育英（5位）**：山田耕介監督がザスパ群馬GMを兼務する歴史的体制。世代交代期で吉田湖太郎・原瑛太郎ら1年生が台頭
- **帝京長岡（6位）**：古沢徹監督が「ボールを握る」を貫く稀有な高体連。U-17代表・木田蓮人がゲームコントロール。鹿島3-2、FC東京1-0などポテンシャル証明済み
- **東京ヴェルディユース（7位）**：「うまいフィルター」の伝統。技術はあるがフィジカル強度が課題

### 下位の構造的課題

- **青森山田（8位）**：絶対王者が16失点の守備崩壊。正木昌宣監督体制での世代交代、対戦相手のメタ戦術浸透が背景。桑原唯斗・深瀬幹太ら選手権優勝メンバーは残る
- **川崎フロンターレU-18（9位）**：U-18代表MF小川尋斗のビルドアップは健在だが、アタッキングサードの精度に課題
- **昌平（11位）**：玉田圭司前監督から芦田徹新監督へ、産みの苦しみ。FC LAVIDA系の才能（佐々木智太郎・白根翼ら）の活用がV字回復の鍵
- **ベガルタ仙台ユース（10位）**・**横浜FCユース（12位）**：残留サバイバル

### Jアカデミー × 高体連の「特性差」

代表選出ポジションを分析すると、**Jアカデミー＝中盤の創造性**（北原槙・小川尋斗）、**高体連＝最終ラインのフィジカル**（メンディーサイモン友）という明確な棲み分けがある。これは育成環境の必然（人工芝の整備された環境 vs トーナメントの極限プレッシャー）。EASTはまさにこの「アカデミー的洗練 vs 高体連的強靭」の異種格闘技戦である。

## 📍 観戦のポイント（WATCHING）

### ① トップ3の異種格闘技戦に注目

「論理的な盾（流経大柏）」vs「代表級の鉾（鹿島ユース）」vs「組織化されたバランス（FC東京U-18）」のスタイル衝突。直接対決でどちらの哲学が機能するかを観るのが醍醐味。特に注目したいのは、**鹿島の代表選手集団に対して流経の堅守がどう対応するか**、**FC東京の中盤北原槙を相手チームがどう封じるか** の2点。

### ② 代表ウィークとコンディション管理

6月以降、AFC U-17アジアカップやJ-VILLAGE CUP U-18で主力が離脱する期間が来る。鹿島は5名同時離脱もあり得る状況で、**スクアッドの深さ（デプス）** が問われる。

ここで医療的視点も持ち合わせて観戦すると面白い：代表選手は移動・連戦による疲労蓄積、戦術成熟の遅れ、そしてバーンアウト（燃え尽き症候群）のリスクと隣り合わせ。試合後半の走行距離低下、給水回数の増加、表情の硬さは、医学的には疲労蓄積のサイン。**ターンオーバーを有効活用できるクラブが、長丁場のリーグ戦を制す** という構図がEASTでは特に顕著に表れる。

### ③ 「選出外」選手のジャイアントキリング

代表選手を多く抱える上位チームと対戦する下位チームの選手にとって、代表選手との直接対決は **「次は自分が選ばれるかもしれない」最高のショーケース**。この強烈なモチベーションが、上位対下位の試合のインテンシティを劇的に跳ね上げ、ジャイアントキリングを誘発する。下位チームの試合こそ「個の覚醒」が観られる瞬間がある。

### ④ 現地観戦の実用情報

EASTは関東圏の会場が多く、公共交通機関でのアクセスが良好。ただし青森山田・ベガルタ仙台ユースのホームゲームは遠征になるため、キックオフ時刻と帰路の最終電車を必ず確認したい。

夏場（6-8月）の試合観戦はWBGT「危険」レベル（31超）に達する日も。観戦者自身も凍らせたペットボトル（首・脇下を冷やす用）、経口補水液、つばの広い帽子、日焼け止めを必ず持参。秋以降は気温の日内変動が大きいため、上着の重ね着スタイルがおすすめ。

## 実装メモ

このファイルを `generate_league_pages.py` の `LEAGUE_DEFS` に以下の形式で組み込み：

```python
LEAGUE_DEFS = [
    (
        "premier-east",                    # スラッグ
        "高円宮杯 JFA U-18 サッカープレミアリーグ EAST",  # 正式名称

        "全国",                            # 地域
        "EAST_DATA_URL",                  # データソース
        TACTICAL_POINTS_EAST,             # ← 新規追加：戦術的見どころ
        WATCHING_POINTS_EAST,             # ← 新規追加：観戦のポイント
    ),
    ...
]
```

各リーグの内容は別ファイル（`league_contents.py` など）に分離する設計が保守性高い。
