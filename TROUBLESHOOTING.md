# トラブルシューティング

## 🚨 古いデータが表示される・5都道府県しか表示されない

この問題は、ブラウザのローカルストレージに古いデータが残っているために発生します。

### 解決方法

#### 方法1: 自動クリア（推奨）
1. **index.htmlを開き直す**
   - 一度ページを閉じて、再度開いてください
   - 初回読み込み時に自動的に古いデータがクリアされます

2. **強制再読み込み**
   - Windows: `Ctrl + Shift + R` または `Ctrl + F5`
   - Mac: `Cmd + Shift + R`

#### 方法2: 手動でローカルストレージをクリア

##### Google Chrome / Microsoft Edgeの場合
1. `F12` キーを押して開発者ツールを開く
2. 上部の **Application** タブをクリック
3. 左側メニューの **Local Storage** を展開
4. サイトのURL（例: `file://` または `localhost`）をクリック
5. 右側に表示される項目を**すべて選択**して削除
   - または右クリック → **Clear** を選択
6. ページを再読み込み（F5）

##### Mozilla Firefoxの場合
1. `F12` キーを押して開発者ツールを開く
2. 上部の **Storage** タブをクリック
3. 左側メニューの **Local Storage** を展開
4. サイトのURLをクリック
5. 右クリック → **Delete All** を選択
6. ページを再読み込み（F5）

##### Safariの場合
1. `Cmd + Option + C` で開発者ツールを開く
2. **Storage** タブをクリック
3. **Local Storage** を選択
4. すべての項目を削除
5. ページを再読み込み（Cmd + R）

#### 方法3: 管理画面から初期化

1. サイトの **管理** ボタンをクリック
2. **データ管理** タブを選択
3. **デフォルトに戻す** ボタンをクリック
4. 確認ダイアログで **OK** をクリック

---

## ✅ 確認方法

正しくデータが読み込まれると：
- **47都道府県すべて**が地域別に表示されます
- 各都道府県がリーグレベルに応じて色分けされています：
  - 🥇 **ゴールド**: プレミアリーグ所属チームあり
  - 🥈 **ブルー**: プリンスリーグ所属チームあり
  - 🥉 **ライトブルー**: 都道府県リーグのみ

## 🔍 その他の問題

### 都道府県をクリックしても反応しない
- JavaScriptエラーがないか確認（F12 → Console タブ）
- ブラウザを最新版に更新

### データが保存されない
- ブラウザがプライベートモード（シークレットモード）になっていないか確認
- ローカルストレージが無効になっていないか確認

### 管理画面でエラーが出る
- 入力内容が正しいか確認（必須項目がすべて入力されているか）
- 数値フィールドに数字以外が入っていないか確認
